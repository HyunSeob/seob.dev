"seob.dev" 는 이현섭의 새로운 기술 블로그입니다. 주로 프론트엔드 개발에 대한 내용을 다룹니다.

## Contribution

오탈자, 잘못된 내용은 Issue로 알려주시거나, 직접 마크다운 원본 파일을 수정하여 Pull Request를 만들어주시면 감사하겠습니다.

## Local Development

```bash
$ yarn
$ yarn re:watch
$ yarn dev # 별도의 프로세스를 사용해야함
```
